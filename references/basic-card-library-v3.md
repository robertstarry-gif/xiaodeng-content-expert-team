# Basic Card Library V3

这是公开版内置的基础卡片库。它不是百科全书，而是一组让安装者立刻写得更像真实中文内容的起步卡片。

使用方式：

- 写作前先选 2-5 张最相关的卡。
- 不要整段照搬卡片，把它们转成当前题目的场景、例子和表达边界。
- 高风险事实仍然需要查证；本卡片库不能替代医学、法律、金融等专业来源。

## 通用场景卡

### SCENE-001 周末反而更累

- 适用：生活方式、情绪、自我管理。
- 场景：周五晚上想着休息，周六醒来却被洗衣、吃饭、回消息、见朋友、补觉这些小选择卡住。
- 可写材料：外卖页面、电影列表、床单、未回消息、朋友邀约。
- 避免：直接说「不是没休息，是不会休息」。

### SCENE-002 下班后不想做决定

- 适用：美食、效率、消费、情绪。
- 场景：晚上回家，连吃什么都要重新比较价格、配送时间、口味和满减。
- 可写材料：外卖满减、预计送达、冰箱里的鸡蛋、剩饭、十分钟热汤。
- 避免：把做饭写成自律或生活美学。

### SCENE-003 消息写了又删

- 适用：情感、职场、沟通。
- 场景：一段话在输入框里改了三遍，最后只发了「嗯嗯」或没有发。
- 可写材料：输入中、撤回、标点、已读、深夜消息。
- 避免：替对方心理下结论。

### SCENE-004 账单让人突然清醒

- 适用：消费、理财、投资教育。
- 场景：月底看账单，不是大额消费吓人，而是很多小钱堆在一起。
- 可写材料：咖啡、会员自动续费、打车、外卖、分期。
- 避免：羞辱消费或给买卖建议。

### SCENE-005 看病前的搜索

- 适用：健康科普。
- 场景：身体不舒服，先搜索，越看越害怕，不知道哪些信息适合自己。
- 可写材料：症状、检查单、药盒、门诊、复诊、红旗信号。
- 避免：诊断、剂量、治疗方案。

### SCENE-006 合同签名前的迟疑

- 适用：法律科普、职场、租房、消费。
- 场景：合同快签了，看到违约、自动续费、解除条件这些条款，才发现自己没看懂。
- 可写材料：押金、期限、违约金、聊天记录、发票、补充协议。
- 避免：替读者制定法律策略。

### SCENE-007 购物车里的犹豫

- 适用：品牌广告、消费决策、测评。
- 场景：东西不贵，但买不买卡住，因为读者担心买回来闲置或不适合。
- 可写材料：购物车、评价区、退货政策、尺寸、使用频率。
- 避免：夸大「必买」「闭眼入」。

### SCENE-008 故事里的沉默

- 适用：小说、故事、情感。
- 场景：人物真正想说的话没有说出口，动作替代解释。
- 可写材料：杯子、门把手、旧衣服、半句消息、停顿。
- 避免：先解释主题，再安排人物行动。

## 坏句卡

### BAD-001 抽象反转句

- 弱句：周末最累的，不是没休息，是一直在判断怎么休息。
- 改法：先给读者进入场景的门。
- 更好：好不容易熬到周末，本想好好放松一下，为什么却更累了？

### BAD-002 假比喻

- 弱句：选项像一群人堵在门口，每个都说自己很重要。
- 问题：画面可爱，但不贴生活。
- 更好：你本来只是想挑个电影，结果又想起衣服没洗、朋友的消息没回、外卖还没点。

### BAD-003 万能时代开头

- 弱句：在当今快节奏的时代，越来越多人开始关注休息。
- 问题：可以放进任何文章。
- 更好：周五晚上你关上电脑，以为这一周终于结束了。

### BAD-004 假平衡

- 弱句：它既是一种选择，也是一种生活态度。
- 问题：漂亮但没信息。
- 更好：它真正改变的不是生活方式，而是今晚不用再被一堆选项拖着走。

### BAD-005 过早升华

- 弱句：这顿饭让你重新掌控人生。
- 问题：小事承受不了大词。
- 更好：它改变不了一周，但至少能把今晚往回拿一点。

### BAD-006 空泛建议

- 弱句：我们要学会保持沟通，建立边界。
- 问题：对，但没法用。
- 更好：先把「你怎么总这样」换成「这件事我有点难受，我想明天再聊」。

### BAD-007 保证式营销

- 弱句：用了它，你的效率会翻倍。
- 问题：没有证据，容易违规。
- 更好：它适合减少整理笔记的重复动作，但最终效率还取决于你的输入习惯。

### BAD-008 医疗确定语气

- 弱句：这种症状说明你已经出现了某种疾病。
- 问题：诊断越界。
- 更好：这种表现可能有多种原因，如果持续、加重或伴随明显异常，应尽快咨询医生。

## 改写动作卡

### MOVE-001 抽象词换动作

- 抽象：选择压力。
- 动作：电影没选完，又切去看外卖；外卖没点，又想起衣服还没洗。

### MOVE-002 大道理换小后果

- 大道理：重新掌控生活。
- 小后果：这顿饭至少让今天不是被外卖列表决定的。

### MOVE-003 人群换一个人

- 泛泛：很多人都会这样。
- 具体：一个忙到周五晚上才松口气的人，很容易在周六上午被一堆小事重新按回去。

### MOVE-004 观点前置改成场景前置

- 弱：真正的休息不是停止工作。
- 强：你明明没有打开电脑，却还是在床上把一天安排了三遍。

### MOVE-005 口号结尾改成收束

- 弱：愿你拥有真正属于自己的生活。
- 强：今晚不需要安排得很好。先把手机放下，把水烧开，就够了。

## 平台卡

### PLATFORM-001 公众号文章

- 开头：真实场景或读者问题先行。
- 中段：每 3-5 段给一个例子或转折。
- 结尾：收住具体问题，不喊口号。
- 避免：标题党、空泛金句、过长清单。

### PLATFORM-002 小红书

- 开头：直接给人群、场景、结果。
- 结构：短段、可截图、步骤明确。
- 语言：口语一点，但不要装熟。
- 避免：过度鸡汤、玄学化、绝对化承诺。

### PLATFORM-003 短视频脚本

- 开头 3 秒：场景或冲突。
- 中段：一个反常识点 + 一个例子。
- 结尾：一句可执行动作。
- 避免：长铺垫、概念堆叠。

### PLATFORM-004 知乎 / 长回答

- 开头：先给结论和适用边界。
- 中段：解释原因、举例、反例、误区。
- 结尾：总结可执行原则。
- 避免：只讲态度，不讲依据。

### PLATFORM-005 落地页 / 广告

- 顺序：痛点 -> 承诺 -> 证据 -> 异议处理 -> 行动。
- 语言：清楚、可信、不要过度兴奋。
- 避免：无证据的「最」「第一」「保证」。

### PLATFORM-006 小说片段

- 开头：人物在做事，不在解释人生。
- 中段：动作、停顿、误会、选择推动。
- 结尾：留下后果或余味。
- 避免：作者替人物讲主题。

## 领域卡

### DOMAIN-001 美食

- 可用材料：火候、时间、香气、口感、清洗、剩菜、食材价格、厨房动线。
- 写法：少写精致生活，多写实际能做。
- 风险：食品安全、过敏、营养功效不要说满。

### DOMAIN-002 情感

- 可用材料：消息、沉默、见面、迟到、争吵后的空白、边界。
- 写法：尊重复杂性，不替任何一方定罪。
- 风险：不要操控、不要诊断人格、不要鼓励危险关系。

### DOMAIN-003 投资 / 金融

- 可用材料：现金流、风险承受力、期限、手续费、波动、流动性。
- 写法：教育优先，条件说清。
- 风险：不提供个性化买卖建议，不保证收益。

### DOMAIN-004 法律

- 可用材料：时间、地点、合同、聊天记录、付款记录、发票、条款。
- 写法：先说适用范围，再说一般原则。
- 风险：不同司法辖区不同，不给个性化法律策略。

### DOMAIN-005 医学 / 健康

- 可用材料：症状持续时间、严重程度、伴随表现、检查、用药史。
- 写法：科普和就医边界并重。
- 风险：不诊断，不开药，不给治疗方案。

### DOMAIN-006 小说

- 可用材料：人物欲望、阻碍、代价、物件、动作、沉默。
- 写法：让人物在场景里露出性格。
- 风险：不要用解释替代戏剧。

### DOMAIN-007 品牌广告

- 可用材料：使用场景、购买顾虑、证据、前后变化、替代方案。
- 写法：把产品放进读者的一天。
- 风险：效果承诺、数据、资质必须有依据。

### DOMAIN-008 职场

- 可用材料：会议、反馈、流程、协作、消息、交付时间。
- 写法：少说成长，多说具体行为。
- 风险：不要鼓吹压榨或不现实的效率神话。

## 标题卡

### TITLE-001 真实问题型

- 适用：公众号、知乎、newsletter。
- 结构：读者熟悉的现象 + 一个真问题。
- 例：为什么你明明休息了，还是觉得累？

### TITLE-002 场景型

- 适用：生活方式、情感、美食。
- 结构：具体时间/动作 + 情绪结果。
- 例：下班后那顿最简单的饭，救的不是胃

### TITLE-003 边界型

- 适用：法律、医学、金融。
- 结构：能说什么 + 不能说什么。
- 例：体检报告有异常，先别急着自己下结论

### TITLE-004 反误区型

- 适用：科普、测评、教育。
- 结构：常见误会 + 更准确的解释。
- 例：别只看收益率，先看你能不能承受这次波动

## 最小使用清单

写任何一篇中文内容前，至少选择：

1. 一张场景卡；
2. 一张坏句卡；
3. 一张平台卡；
4. 一张领域卡；
5. 一张改写动作卡。

如果选不出来，说明方向还不够具体，应先由小灯主编补 brief。
