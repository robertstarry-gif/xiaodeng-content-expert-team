# Platform And Format Guide

Use this reference when adapting one idea across channels or when the user asks for a platform-specific artifact.

## Blog / Newsletter / Essay

- Put the value or thesis early.
- Build trust with examples, evidence, and trade-offs.
- Use sections that answer reader questions.
- End with a practical takeaway, next step, or memorable closing thought.

## X / LinkedIn / Feed Thread

- Start with a clear hook or claim.
- Use short units, strong sequencing, and one idea per post.
- Make the save/share value obvious.
- Avoid long setup before the point.

## Short Video

- Start with a scene, conflict, or surprising useful claim.
- Keep one core tension.
- Use spoken language.
- End with a concrete conclusion or next action.

## 小红书 / Instagram Carousel

- Treat each card as one job.
- Use fewer branches and clearer labels.
- Prefer checklists, choices, before/after, or mistake/fix patterns.
- Keep captions supportive, not redundant.

## Landing Page / Ad

- Name the audience and pain.
- Make one promise.
- Give proof or reason to believe.
- Handle the main objection.
- End with one clear call to action.

## Fiction

- Start inside a charged situation.
- Let character desire create movement.
- Use concrete images and actions.
- Let subtext carry emotion where possible.
- Avoid explaining the whole theme too early.
